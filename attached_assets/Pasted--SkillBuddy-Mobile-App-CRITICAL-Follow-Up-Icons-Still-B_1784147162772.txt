**Systematic debugging steps to actually find the root cause this time:**
1. Check which icon package is installed (`package.json`) — confirm it's actually installed (`node_modules` present) and the version is compatible with the installed Expo SDK version. A version mismatch between `expo`, `react-native`, and the icon package is a very common cause of silent icon failures.
2. If using `@expo/vector-icons`: these require no extra font-loading in modern Expo projects, but confirm the import syntax is correct (e.g. `import { Ionicons } from '@expo/vector-icons'` and usage `<Ionicons name="home" size={24} color="#4F9D82" />` with a **valid icon name** from that specific icon set — an invalid/misspelled `name` prop silently renders nothing rather than throwing a visible error, which is a very common cause of "icons just don't show up").
3. If using `lucide-react-native`: confirm it's imported per-icon correctly (e.g. `import { Home } from 'lucide-react-native'`) and rendered as a component (`<Home size={24} color="#4F9D82" />`), and that the `lucide-react-native` peer dependency `react-native-svg` is installed (this library requires `react-native-svg` to render at all — **this is a very likely missing piece** if icons are completely invisible app-wide rather than just individually broken).
4. Do a completely clean reinstall and cache clear: delete `node_modules` and lockfile, reinstall dependencies, then start with `expo start -c` (clears Metro cache) to rule out stale/corrupted bundling.
5. Temporarily render a single icon in complete isolation on a blank test screen with a hardcoded color and size (no theme tokens, no dynamic props) to confirm whether the icon library works at all in this project — if even that fails, the issue is fundamental (missing dependency/setup); if that works but icons elsewhere don't, the issue is in how icons are being invoked elsewhere (likely broken theme-token color values passed as `color` props, per the hypothesis from the last phase).
6. Fix whichever root cause is found, then re-verify all icons across the whole app (nav bar, back arrows, and every screen-level icon) are visible before moving to any other task in this prompt.
