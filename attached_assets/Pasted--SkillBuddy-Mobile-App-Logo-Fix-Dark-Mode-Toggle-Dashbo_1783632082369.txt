This is a key premium feature for the app.

**Required behavior:**
- Add a theme toggle control accessible from the app (e.g. in Settings or as a quick-access icon in the header/profile area — pick whichever placement fits the existing screen designs best) that lets the user manually switch between Light and Dark theme, in addition to (or instead of, if simpler) following the system theme.

This should feel premium and intentional — reference how well-polished apps (e.g. Twitter/X's theme switch animation) handle this kind of transition, and replicate that quality level using `react-native-reanimated` (e.g. an animated overlay view that expands/wipes across the screen while the underlying theme value swaps underneath it).
- Persist the user's theme choice (e.g. via `expo-secure-store` or `AsyncStorage`, since this isn't sensitive data) so it's remembered across app restarts.
- Test the toggle from multiple screens (Home, Profile, a detail screen) to confirm the animation and color swap work consistently everywhere, not just from one entry point.
